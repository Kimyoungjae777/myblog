# 생활코딩(변수)-20,21,22,23

### 변수

- 영어로는 Variable 로 (문자나 숫자 같은) 데이터를 담는 컨테이너다. 여기에 담겨진 데이터는 다른 데이터로 바꿀 수 있다. 변수는 마치(사람이 쓰는 언어인) 자연어에서 대명사와 비슷한 역할을 한다 .

### 정수와 실수

- 

```java
int d;
		d=a+c;
		System.out.println(d);
```

- output → error ⇒ int 에 int+double 을 하니까 type mismatch 가 일어난다.
